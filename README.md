### Installation
```sh
# Clone the repository
git clone https://github.com/parmeeta2/ecommerce.git
cd ecommerce

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install
```

### Run the Application
```sh
# Start the backend server
cd server
npm start

# Start the frontend client
cd ../client
npm start
```
